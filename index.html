About Me

If you are looking for someone who is going to sit in meetings or in an interview and talk technical terms then you've come to the wrong place but If you are looking for a hard working engineer that can deliver solutions and fix problems keep reading. I built this digital portfolio for a simple demonstrate some of the skills I know. The back end API is built with Java Spring Boot providing some basic rest endpoints and is hosted on Heroku while the front end is built using Angular 12 and is hosted on Netlify. Angular Universal is used to load some of the resources server side to speed things up and help increase the SEO score. There is a service worker that is used to control caching and turn this bad boy into a Progressive Web App (Try installing on your mobile device). You can run a lighthouse scan in the chrome dev tools and see how high it scores. Most of the stuff in this app is pretty basic so if there is a specific skill or feature you would like to see or just recommend please just shoot me a message at misterhedrick@gmail.com and I will be glad to take a look. I’m always looking to learn more and expand my skill set!!
